Beschreibung
************

Random Data File Creator, kurz RDFC, ist eine Konsolenanwendung für Microsoft Windows, die binäre Dateien beliebiger Größe erstellt und mit Zufallszahlen beschreibt. Damit kann man als Entwickler z.B. sehr große Dateien (bspw. 1 TB) erstellen, um zu testen, wie die eigene Anwendung damit umgeht.

Anwendung
*********

Sie können die Anwendung rdfc.exe in einer Eingabeaufforderung (Konsolenfenster unter Windows), dem Dialog Ausführen… oder aus einer beliebigen anderen Anwendung startet, die es erlaubt, Startparameter zu übergeben. Folgende Startparameter sind möglich:

rdfc.exe <file> <size> [<unit> [<overwrite>]]

<file> steht für Pfad und Name der Datei, die erzeugt werden soll.

<size> steht für die Dateigröße. Sie können eine beliebige Zahl größer 0 angeben. Beachten Sie, dass die Zahl in der als <unit> angegebenen Einheit interpretiert wird.

<unit> steht für die Einheit der Dateigröße. Möglich sind GB, MB, kB und B, wobei B der Standardwert ist.

<overwrite> ermöglicht es, existierende Dateien zu überschreiben. Geben Sie dafür als vierten Startparameter ‚overwrite’ an. Andernfalls bricht die Anwendung ab, sofern die in <file> angegebene Datei schon existiert.

Hinweis: Die Anwendung prüft nicht, ob für die Erstellung der Datei genügend Platz auf der Festplatte (bzw. dem angegebene Medium) vorhanden ist. Auch die Beschränkung der  maximalen Dateigröße durch Dateisysteme wird nicht geprüft. Die Anwendung bricht im Fehlerfall mit einer entsprechenden Meldung ab.

Systemvoraussetzungen
*********************

Microsoft Windows 98/ME/2000/XP/2003
Ausreichen Platz auf der Festplatte

Download und Installation
*************************

Laden Sie das ZIP-Archiv herunter (http://www.bertel.de/software/rdfc) und entpacken Sie es in einen beliebigen Ordner auf Ihrer Festplatte. Eine Installation ist nicht notwendig.
